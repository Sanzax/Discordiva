# Discordiva

Discord Rich Presence integration for **Hatsune Miku: Project DIVA Mega Mix+**. This mod displays detailed gameplay information directly on your Discord profile, enabling others to see what you are playing in real time.

---

## Features

The mod provides granular, real-time Discord Rich Presence data including:

* **Song Difficulty** and **Star Rating**
* **Song Name**
* **Artist Name**
* **Game State** detection (Shown as a small image):
  * In menu
  * Playing
  * Paused
  * Clear
  * Not clear
* **Game Mode** detection:
  * Rhythm Game (default; not shown as a labeled mode)
  * Music Video
  * Practice Mode
* **Song Settings**:

  * No Fail Mode
  * Game Modifiers: HI SPEED, HIDDEN, SUDDEN
* **Playback Progress**:

  * Current timestamp in song
  * Total song length
* **Gameplay Metrics**:

  * Clear Percentage
  * Score
  * Current Combo

---

## Requirements

* **Hatsune Miku: Project DIVA Mega Mix+**
* **Discord Desktop Application** running while the game is active
* [**DIVA Mod Loader**](https://github.com/blueskythlikesclouds/DivaModLoader)

---

## Installation
1. Install **DIVA Mod Loader** if you haven't already
2. Download the latest release
3. Unzip the file
4. Place the Discordiva mod folder inside your mods folder
